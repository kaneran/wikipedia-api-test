To test this out, please follow the below steps:
1) Clone this repository
2) Open a terminal and access the folder for this repository and from there, run command 'npm i'
3) To run the automated unit tests, run command 'npm run test'
4) To run the REST api server, run command 'npm run start', then test any of the endpoints via http://localhost:3000

NOTE: To run the npm command, please ensure Node.js is installed
